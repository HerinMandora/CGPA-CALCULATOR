# CGPA Calculator

A web-based CGPA calculator built to help students calculate semester and overall CGPA.

## Features
- CGPA calculation based on user input
- JavaScript-based logic
- Simple and user-friendly interface

## Tech Stack
- JavaScript
- HTML
- CSS

SEE (Semester-End Exam)

This calculator helps ITNU ECE students quickly determine:

Their current CE%

Their final subject %

The SEE marks required to reach a target percentage

Expected CGPA from marks

A SEE Prediction Table (40,50,60,70,80,90,100)

Full Semester CGPA based on subject marks
